# Quick Start Guide for Trainees

## Before You Begin
- You need a Google Gemini API key (get free from https://aistudio.google.com/)
- A modern web browser (Chrome, Firefox, Safari, Edge)
- About 45 minutes for full training

## Step 1: Get Your API Key
1. Go to https://aistudio.google.com/
2. Click **"Create API Key"** button
3. Copy the key (looks like: `AIzaSyD8...`)
4. Keep this **secret** - don't share it

## Step 2: Open the Training
1. Open **trainer.html** in your browser
2. Or visit: https://github.com/cmc-creator/NyxCodex → Clone/Download
3. Works offline after opening (but needs API key for AI features)

## Step 3: Enter Your API Key
1. Navigate to **Slide 8** (Interactive Crisis Lab)
2. See the "API Key:" field at the top right
3. Paste your key there
4. Click elsewhere to save

## Step 4: Start Training!
1. **Slides 1-6**: Learn core de-escalation concepts
   - Body language
   - Grounding techniques
   - Verbal skills
   - Strategic environment
   
2. **Slide 7**: Diagnosis Library
   - Browse 8 psychiatric diagnoses
   - Learn specific triggers & responses
   - Read de-escalation playbooks
   
3. **Slide 8**: Interactive Crisis Lab 🎮
   - **New Crisis**: Generate realistic scenarios
   - Select diagnosis type
   - Type your response
   - Get instant AI scoring (A-F)
   - Earn XP!
   
4. **Slide 9**: Clinical Debrief
   - Describe a real incident
   - Get mentor feedback
   - Learn from experience
   
5. **Slide 10**: Quick Reference
   - Dos and don'ts checklist
   
6. **Slide 11**: Your Certificate
   - Download PDF certificate
   - Shows: scenarios completed, average score, XP earned

## Keyboard Shortcuts
- **Left/Right Arrow**: Navigate slides
- **A**: Open Admin Dashboard (see team scores, export data)
- **F12**: Dev console (if you need to debug)

## How You Earn Points
- **Complete Scenario**: 25-250 XP (based on A-F score)
- **Build Script**: 50 XP
- **Get Debrief**: 100 XP

## Tips for Best Results
✅ Practice regularly (15-30 min sessions better than cramming)  
✅ Vary diagnoses to learn different presentations  
✅ Read other staff's responses in leaderboard  
✅ Use the debrief feature after real incidents  
✅ Download your certificate when done  

## Troubleshooting

**"API Error"**
- Paste your key again - it might have been cut off
- Check that you have remaining API quota at https://aistudio.google.com/

**"Scenario won't load"**
- Refresh the page
- Make sure internet connection is active
- Try a different diagnosis type

**"My progress disappeared"**
- Check if you're in Private/Incognito mode (disables localStorage)
- Progress saves to this browser only
- Ask admin to restore from backup export

**"Can't download certificate"**
- Make sure pop-ups aren't blocked
- Try a different browser
- Check your Downloads folder

## What Happens With Your Data?
- ✅ Your progress is saved to your browser automatically
- ✅ Your facility admin can export data for team tracking
- ✅ You can download your certificate anytime
- ⚠️ Data is tied to YOUR BROWSER - if you use a different device, it's fresh start
- 💡 Tip: If trained on shared computer, ask admin to backup your data
